## バッチ処理におけるFormとEntityの使い分け

- 入力データを**単一テーブルに出力**する場合: Formを作らずEntityのみで処理する
- 入力データを**複数テーブルに出力**する場合: Formを作成して処理対象データをFormデータで引き回す
