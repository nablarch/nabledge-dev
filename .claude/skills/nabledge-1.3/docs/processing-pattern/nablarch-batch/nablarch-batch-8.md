# バッチ処理を実行するとProcessStop(kill process.)が発生し、処理が異常終了してしまいます。対処方法を教えてください

> **question:**
> バッチ処理を実行すると、以下のメッセージがログに出力されてバッチ処理が異常終了してしまいます。
> 対処方法を教えてください。

> ```java
> [50 ProcessStop] kill process.
> ```

> **answer:**
> Nablarchでは、実行中のプロセスを強制終了する機能を提供しています。
> この機能を使用して処理を強制停止した場合、データベースの停止フラグのクリア処理は行われないため、
> 停止フラグの値をクリアする必要があります。

> 詳細は、以下のドキュメントを参照してください。

> * >   **[Nablarch Application Framework 解説書]** -> **[リファレンス]** -> **[ハンドラリファレンス]** -> **[プロセス停止制御ハンドラ]**
