# ファイル読み込み処理で、想定するレコード識別子以外の値がファイルに存在した場合の動作を教えてください

> **question:**
> 例えば、レコード区分が **1,2,8,9** の何れかであるファイルの場合で、
> これら以外の値がレコード区分に設定されていた場合、ファイル読み込み処理はどのような動作となるのでしょうか？

> **answer:**
> ファイル読み込み処理では、ファイルのフォーマット定義ファイルに定義されている識別子以外の値が存在した場合には、例外を送出して即処理を終了します。

> ※例外を送出するため、バッチの処理は異常終了となります。(FATALログが出力され、0以外の値を終了コードとしてJavaプロセスが終了します。)
