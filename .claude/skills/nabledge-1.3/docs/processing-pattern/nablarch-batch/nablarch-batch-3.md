# ファイル読み込み処理で、想定するレコード識別子以外の値がファイルに存在した場合の動作を教えてください

## 想定外レコード識別子が存在した場合の動作

ファイル読み込み処理では、フォーマット定義ファイルに定義されていない識別子の値がファイルに存在した場合、例外を送出して即処理を終了する。

> **重要**: 例外送出によりバッチは異常終了となる。FATALログが出力され、0以外の終了コードでJavaプロセスが終了する。
