# codeSelectやselectタグで、先頭に空白要素を追加する方法を教えてください

## withNoneOptionによる空白要素の追加

`withNoneOption` を使用することで、codeSelectタグやselectタグの先頭に空白要素を追加できる。

空白要素を選択した場合、リクエストパラメータの値として空文字列(`""`)が送信される。
