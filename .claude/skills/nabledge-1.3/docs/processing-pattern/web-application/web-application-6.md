# 画面に表示するエラーメッセージを途中で改行することは出来ますか?

## 画面表示メッセージの改行方法

メッセージに `<br />` を含めてもサニタイジングされるため、HTMLタグとして機能せず `<br />` がそのまま画面表示される。

メッセージに物理的な改行を含めること。taglibは表示対象データに改行が含まれている場合、`<br />` タグに置き換えて出力する。

データベースでメッセージを管理する場合、以下のようにCHR(13) || CHR(10)（`\r\n`）を使って改行を含むデータを登録する：

```sql
-- CHR(13) || CHR(10)は、「\r\n」を表す
INSERT INTO MESSAGE (MESSAGE_ID, MESSAGE_TEXT) VALUES ('ID01', 'line 1' || CHR(13) || CHR(10) || 'line2');
```
