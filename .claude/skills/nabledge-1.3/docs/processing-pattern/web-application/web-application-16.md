## input(type="hidden")タグの出力方法

`n:hidden`タグはHTMLタグを出力せず、ウィンドウスコープに値を出力する。HTMLの`input(type="hidden")`タグを出力してJavaScriptから操作する場合は、`n:plainHidden`タグを使用する。

```jsp
<%-- n:plainHiddenタグの使用例 --%>
<n:plainHidden name="user.id" />
```
