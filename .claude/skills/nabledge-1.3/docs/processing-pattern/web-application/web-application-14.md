# Firefoxで2重サブミット防止が効かないことがあるようなのですが?

> **question:**
> 二重サブミット防止が効いているかどうかを確認するために、n:submitLinkで生成されたリンクを
> 何度か素早くダブルクリックしてみたのですが、
> まれにエラー画面が表示されます。
> サーバ側のログを確認すると、どうやら2回連続でリクエストが送信されてしまうことがあるようです。

> この現象は Firefox でのみ発生し、Chrome/Safari/IE6 では発生しません。

> **answer:**
> Firefoxの古いバージョン(3.x系を含む)で発生する事象です。
> 比較的最近のバージョン(9.01以降)にアップデートしていただくことで解消します。

> また、二重サブミット防止機能は、サーバサイドのトークンチェックと併用することが前提となっているので、
> 本事象が発生しても業務処理が2重実行されるようなことはありません。
