# 子画面を開く際に画面サイズや、属性を指定する事は出来ますか?

## 子画面（サブウィンドウ）の画面サイズ・属性指定

`popupSubmit`、`popupButton`、`popupLink` タグで子画面（サブウィンドウ）を開く際、`popupOption` 属性を指定することで画面サイズや任意の属性を変更できる。
