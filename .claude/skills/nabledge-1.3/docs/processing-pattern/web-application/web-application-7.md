# コード値精査時に指定するパターンには何を指定するのでしょうか？

## コード値精査（CodeValue）のパターン属性指定

`CodeValue` の `pattern` 属性に指定するパターン値については、[../all/3](../../about/about-nablarch/about-nablarch-3.md) を参照。なお、`pattern` 属性に数値（例: `2`）を直接指定すると「パターンが存在しない」エラーが発生する。パターンの指定方法はドキュメントで確認すること。
