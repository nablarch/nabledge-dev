# 精査エラーのメッセージを、任意の項目に対応させて画面表示させる方法を教えてください

## ValidationUtil.createMessageForPropertyを使ったエラーメッセージの任意項目への表示

データベースアクセスを必要とする精査など、EntityやForm以外の箇所で精査を行った場合のエラーメッセージを任意の画面項目と対応させて表示するには、`ValidationUtil.createMessageForProperty` を使用する。

エラー発生箇所を示すプロパティ名をメッセージの属性に追加できるため、エラーメッセージを特定の項目に対応させて表示したり、画面の任意の箇所にメッセージを表示させたりすることができる。
