# Nablarch Toolbox

## Nablarch Toolbox ツール一覧

Nablarch Toolboxには試作段階のツールが含まれる場合がある。

| ツール | 概要 |
|---|---|
| :ref:`entity_generator` | Formを自動生成する |
| :ref:`jsp_generator` | HTMLファイルを基にJSPファイルを自動生成する |
| :ref:`config_generator` | 環境設定ファイルを自動生成する |
| [SQL自動生成ツール](../../../knowledge/development-tools/toolbox/assets/toolbox-tool/SQL自動生成ツール.xls) | テーブル定義書を基にSQL文を自動生成する |
| [排他制御主キークラス自動生成ツール](../../../knowledge/development-tools/toolbox/assets/toolbox-tool/排他制御主キークラス自動生成ツール.xls) | テーブル定義書を基に排他制御主キークラスを自動生成する。本ツールを使用するには [物理型・Javaオブジェクトマッピング定義](../../../knowledge/development-tools/toolbox/assets/toolbox-tool/物理型・Javaオブジェクトマッピング定義.xls) が必要 |
| [フォーマット定義ファイル自動生成ツール](../../../knowledge/development-tools/toolbox/assets/toolbox-tool/フォーマット定義ファイル自動生成ツール.xls) | 外部インタフェース設計書からFWが使用するフォーマット定義ファイルを自動生成する |
| [リクエストテーブル登録用データ作成ツール](../../../knowledge/development-tools/toolbox/assets/toolbox-tool/リクエストテーブル登録用データ作成ツール.xls) | リクエストID一覧からリクエストテーブルに登録するマスタデータのテーブル投入用シートを自動生成する |
| [コードテーブル登録用データ出力ツール](../../../knowledge/development-tools/toolbox/assets/toolbox-tool/コードテーブル登録用データ出力ツール.xls) | コード設計書からコードテーブルに登録するマスタデータのテーブル投入用シートを自動生成する |
| [メッセージテーブル登録用データ作成ツール](../../../knowledge/development-tools/toolbox/assets/toolbox-tool/メッセージテーブル登録用データ作成ツール.xls) | メッセージ設計書からメッセージテーブルに登録するマスタデータのテーブル投入用シートを自動生成する |
| [shell_generator](toolbox-ShellGenerator.md) | ジョブスケジューラから起動されるShell Scriptと環境依存項目を設定したファイルを自動生成する |
| [テーブル定義書及びドメイン定義書出力手順](../../../knowledge/development-tools/toolbox/assets/toolbox-tool/テーブル定義書およびドメイン定義書出力手順.xls) | ObjectBrowserのレポート出力機能を使ってテーブル定義書およびドメイン定義書を出力する手順を説明したドキュメント |
| [テーブル定義書ドメイン定義書 整合性チェックツール](../../../knowledge/development-tools/toolbox/assets/toolbox-tool/テーブル定義書_ドメイン定義書_整合性チェックツール.xls) | テーブル定義書記載のドメインがドメイン定義書に定義されていることや、桁数、データ型の整合性をチェックする |
| [published_api](toolbox-PublishedApiConfigGenerator.md) | 使用不許可APIチェックツールの設定ファイルとなる公開APIの一覧と、公開APIのJavadocを生成する |
