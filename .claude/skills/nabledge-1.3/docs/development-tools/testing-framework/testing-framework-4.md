# Excelのテストデータに設定した整数値が、テスト実行時には小数部ありの数値となってしまいます。回避方法を教えてください

## ExcelテストデータのセルをすべてString書式に設定する

ExcelのAGE列など数値データを標準/数値書式で設定した場合、テスト実行時に小数（例: `20` → `20.0`）に変換されるため、テストが正常に実施できない。

> **重要**: Excelに記述するテストデータの書式はすべて**文字列書式**で設定すること。文字列書式以外（標準や数値書式）の場合、自動テストFWがデータを正しく読み取れない。

詳細は「[Nablarchプログラミング・単体テストガイド] → [自動テストフレームワークの使用方法] → [自動テストフレームワーク] → [セルの書式]」参照。
