# コード名称やコード存在チェック時に指定するパターンには、何を指定したらいいのでしょうか？

## コード機能のpattern属性に指定する値

## コード機能のpattern属性に指定する値

`pattern` 属性には、コードテーブルの対応するカラムの**物理名**を設定する必要がある。

> **警告**: パターンの番号（例: `2`）を指定してもエラーになる。カラムの物理名（例: `PATTERN2`）を指定すること。
