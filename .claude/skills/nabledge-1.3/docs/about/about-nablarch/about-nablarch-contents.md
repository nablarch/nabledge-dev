## Nablarch の製品構成

Nablarchの製品構成について説明します。

| コンテンツ名 |  | 説明 |
|---|---|---|
| Nablarchライブラリ | Nablarch Application Framework | Java実行環境上で実行可能なアプリケーション実行基盤です。 下記が含まれます。  * オブジェクトコード * APIドキュメント * 解説書 |
|  | Nablarch Testing Framework | 自動テストフレームワークや静的解析ツール、テスト支援ツール を含むテスティングフレームワークです。 下記が含まれます  * オブジェクトコード * APIドキュメント |
| Nablarchガイド | 開発ガイド | Nablarch ライブラリを使用してアプリケーションの実装とテストを実施する方 法を説明したドキュメントです。 初期学習用のチュートリアルも含まれます。 |
|  | 環境構築ガイド | Nablarch ライブラリを使用してアプリケーション開発を行う際の開発環境及び 開発リポジトリ構築方法について説明したドキュメントです。 |
|  | チュートリアル用コンテンツ | 業務アプリケーションの実装方法を説明するための サンプルアプリケーションです。 ソースコードは Eclipse のワークスペースの形式となっています。 |
| Nablarch開発標準 | 設計標準 | 設計時の指針となる標準です。 |
|  | ドキュメント規約 | 設計書を記載する際のルールを定めた規約です。 |
|  | 設計書フォーマット | 設計書のフォーマットとサンプルです。 アプリケーションの外部設計、内部設計に必要な書式は全て揃っています。 |
|  | コーディング規約 | 開発に必要となる各種コーディング規約です。 |
|  | 単体テスト | 単体テストケースやテスト観点の標準です。 |
| Nablarchサンプル | 業務機能サンプル | 実プロジェクトで頻繁に作成する機能のサンプルアプリ ケーションです。 (ログイン機能、ユーザ管理機能等) |
|  | 拡張モジュールサンプル | Nablarchの拡張モジュールのサンプルです。 |
| Nablarch Toolbox |  | プログラムやテストデータの自動生成ツールなど、アプリケーション開発の品質 と生産性を向上するツール群です。 |

各コンテンツについて、カスタマイズの可否、ソースコードの開示・非開示、特定のOSSやミドルウェアへの依存の有無を説明します。

| コンテンツ名 |  | カスタマイズ可否 [1] | ソースコード開示 [2] | OSSの利用 [3] | 特定M/Wへの依存 [4] |
|---|---|---|---|---|---|
| Nablarchライブラリ | Nablarch Application Framework | 不可 | 非開示 | なし | なし |
|  | Nablarch Testing Framework | 不可 | 非開示 | あり | なし |
| Nablarchガイド | 開発ガイド | 可 | － | － | － |
|  | 環境構築ガイド | 可 | － | あり | あり |
|  | チュートリアル用コンテンツ | 可 | 開示 | あり | あり |
| Nablarch開発標準 | 設計標準 | 可 | － | － | － |
|  | ドキュメント規約 | 可 | － | － | － |
|  | 設計書フォーマット | 可 | － | － | － |
|  | コーディング規約 | 可 | － | － | － |
|  | 単体テスト | 可 | － | － | － |
| Nablarchサンプル | 業務機能サンプル | 可 | 開示 | あり | あり |
|  | 拡張モジュールサンプル | 可 | 開示 | あり | あり |
| Nablarch Toolbox |  | 可 | 開示 | あり | あり |

お客様による改変・翻案が可能かどうか。

ソフトウェアの場合に、ソースコードを開示するかどうか。ソースコード非開示のコンテンツについては、オブジェクトコードを納品します。

オープンソースソフトウェアを使用している、または今後使用する可能性があるか否か。
ドキュメントの場合は、オープンソースソフトウェアを使用することを前提とした記述がある、または今後記述する可能性があるか否か。
本番環境で稼働させるソフトウェアについては、オープンソースソフトウェアを使用しない方針としています。

特定のミドルウェアに依存している、または今後依存する可能性があるか否か。
ドキュメントの場合は、特定のミドルウェアを使用することを前提とした記述がある、または今後記述する可能性があるか否か。
