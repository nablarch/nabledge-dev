## Nablarchの製品構成

Nablarchのコンテンツ一覧:

| コンテンツ名 | サブコンテンツ | 説明 |
|---|---|---|
| Nablarchライブラリ | Nablarch Application Framework | Java実行環境上で実行可能なアプリケーション実行基盤。オブジェクトコード・APIドキュメント・解説書を含む。 |
| | Nablarch Testing Framework | 自動テストフレームワーク・静的解析ツール・テスト支援ツールを含むテスティングフレームワーク。オブジェクトコード・APIドキュメントを含む。 |
| Nablarchガイド | 開発ガイド | Nablarchライブラリを使用したアプリケーション実装・テスト方法を説明したドキュメント。初期学習用チュートリアルも含む。 |
| | 環境構築ガイド | アプリケーション開発における開発環境・開発リポジトリ構築方法を説明したドキュメント。 |
| | チュートリアル用コンテンツ | 業務アプリケーションの実装方法を説明するサンプルアプリケーション。Eclipseワークスペース形式。 |
| Nablarch開発標準 | 設計標準 | 設計時の指針となる標準。 |
| | ドキュメント規約 | 設計書を記載する際のルールを定めた規約。 |
| | 設計書フォーマット | 設計書のフォーマットとサンプル。外部設計・内部設計に必要な書式を含む。 |
| | コーディング規約 | 開発に必要となる各種コーディング規約。 |
| | 単体テスト | 単体テストケースやテスト観点の標準。 |
| Nablarchサンプル | 業務機能サンプル | 実プロジェクトで頻繁に作成する機能のサンプルアプリケーション（ログイン機能、ユーザ管理機能等）。 |
| | 拡張モジュールサンプル | Nablarchの拡張モジュールのサンプル。 |
| Nablarch Toolbox | — | プログラムやテストデータの自動生成ツールなど、アプリケーション開発の品質と生産性を向上するツール群。 |

各コンテンツのカスタマイズ可否・ソースコード開示・OSS利用・特定M/W依存:

| コンテンツ名 | サブコンテンツ | カスタマイズ可否 | ソースコード開示 | OSSの利用 | 特定M/Wへの依存 |
|---|---|---|---|---|---|
| Nablarchライブラリ | Nablarch Application Framework | 不可 | 非開示 | なし | なし |
| | Nablarch Testing Framework | 不可 | 非開示 | あり | なし |
| Nablarchガイド | 開発ガイド | 可 | — | — | — |
| | 環境構築ガイド | 可 | — | あり | あり |
| | チュートリアル用コンテンツ | 可 | 開示 | あり | あり |
| Nablarch開発標準 | 設計標準 | 可 | — | — | — |
| | ドキュメント規約 | 可 | — | — | — |
| | 設計書フォーマット | 可 | — | — | — |
| | コーディング規約 | 可 | — | — | — |
| | 単体テスト | 可 | — | — | — |
| Nablarchサンプル | 業務機能サンプル | 可 | 開示 | あり | あり |
| | 拡張モジュールサンプル | 可 | 開示 | あり | あり |
| Nablarch Toolbox | — | 可 | 開示 | あり | あり |

注釈:
- カスタマイズ可否: お客様による改変・翻案が可能かどうか
- ソースコード開示: ソフトウェアの場合にソースコードを開示するかどうか。非開示の場合はオブジェクトコードを納品
- OSSの利用: オープンソースソフトウェアを使用している、または今後使用する可能性があるか否か。ドキュメントの場合は、オープンソースソフトウェアを使用することを前提とした記述がある、または今後記述する可能性があるか否か。本番環境で稼働させるソフトウェアについてはOSSを使用しない方針
- 特定M/Wへの依存: 特定のミドルウェアに依存している、または今後依存する可能性があるか否か。ドキュメントの場合は、特定のミドルウェアを使用することを前提とした記述がある、または今後記述する可能性があるか否か
