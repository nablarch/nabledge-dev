# マスタデータのセットアップ

**公式ドキュメント**: [マスタデータのセットアップ]()

## マスタデータのセットアップを行うAntターゲット

> **注意**: マスタデータとは、コードテーブル・メッセージテーブル等のプロジェクト共通データ、認可チェック機能・採番機能に必要なデータを指す。多くはプロジェクト共通データとして提供済みのため個別追加不要だが、「認可チェック機能に必要なデータ」はプログラマが追加・修正する運用のプロジェクトもある。開発時にマスタデータの編集が必要か否かはプロジェクトのアーキテクトに確認すること。

> **警告**:
> - 開発時はマスタデータがDBに登録されていることを前提に作成すること。単体テストで改めて定義しないこと。
> - マスタデータが更新されたらすぐに取り込むこと。

| Antビルドファイル | 実行するターゲット |
|---|---|
| `tool/db/db_setup-build.xml` | DB初回セットアップ（2回目以降は「DB再セットアップ」） |

## Eclipseからマスタデータセットアップツールを実行する

EclipseからAntターゲットを実行する手順:

1. Antビューを表示する
2. Antビューの「ビルドファイルの追加」を選択する
3. 追加するビルドファイルに `tool/db/db_setup-build.xml` を選択する
4. マスタデータセットアップのターゲットを実行する

初回セットアップ後は、Antビューに登録された「DB再セットアップ」ターゲットを実行することで、データベースにマスタデータが反映される。
