# ユーザ登録情報電文受信処理の仕様

## 機能概要

外部システムからの送信電文を受信し、ユーザ登録情報受信テーブルにレコードを登録する。後続処理（常駐バッチ）が一定時間間隔でテーブルを監視し、未処理データが存在する場合にユーザ情報を登録する。

**INPUT**: ユーザ登録情報電文  
**OUTPUT**: ユーザ登録情報受信テーブル
