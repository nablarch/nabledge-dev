# 変更履歴

nabledge-1.3プラグインの主な変更内容を記録しています。

フォーマットは [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) に基づいています。

## [0.2] - 2026-05-26

### 変更
- 知識ファイルの生成方式をAIによる要約からルールベース変換に切り替えました。AI特有のハルシネーションが発生しなくなり、ミッションクリティカルなシステムでも安心して利用できるようになりました。
- ワークフロー実行中にスクリプトの許可プロンプトが表示されなくなりました。作業の中断なくスムーズにワークフローを実行できます。
- 質問回答のフローを見直し、処理方式や目的に応じてより適切な回答ができるようになりました。

### 修正
- 閲覧用ドキュメント内のリンクが正しく表示されない問題を修正しました。
- コード分析のシーケンス図にJavaメソッド名が正しく出力されるようになりました。実際のコードと対応した図を確認できます。
- コード分析の処理フローにヘルパー/プライベートメソッドが含まれるようになりました。より詳細な処理の流れを把握できます。
- コード分析の出力先ファイルパスが結果に明示されるようになりました。出力ファイルをすぐに確認できます。

### 追加
