# ユーザ登録情報電文受信処理の仕様

## 

外部システムからの送信電文を入力として、ユーザ登録情報受信テーブルにレコードを登録する。

![ユーザ登録情報電文受信処理フロー](../../../knowledge/guide/mom-messaging/assets/mom-messaging-01_userRegisterMessageReceiveSpec/userRegisterMessageReceive.png)

ユーザ登録情報受信テーブルは、後続処理（常駐バッチ）によって一定時間間隔で監視され、未処理のデータが存在する場合にユーザ情報を登録する。

**処理概要**:
1. ユーザ登録情報電文受信処理: ユーザ登録情報電文を受信し、データレコードの内容をユーザ登録情報受信テーブルに登録する。
   - INPUT: ユーザ登録情報電文
   - OUTPUT: ユーザ登録情報受信テーブル
