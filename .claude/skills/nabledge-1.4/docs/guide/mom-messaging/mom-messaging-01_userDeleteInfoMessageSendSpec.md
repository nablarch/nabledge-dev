# ユーザ削除情報電文送信処理の仕様

## 

![](../../../knowledge/guide/mom-messaging/assets/mom-messaging-01_userDeleteInfoMessageSendSpec/userDeleteInfoMessageSend.png)

ユーザ削除情報メッセージ送信テーブルを入力として外部システムに電文を送信する。

> **注意**: 送信対象のユーザ削除情報は、前段処理のユーザ情報削除バッチで登録される。ユーザ情報削除バッチの仕様は [../../04_Explanation_batch/01_userDeleteBatchSpec](../nablarch-batch/nablarch-batch-01_userDeleteBatchSpec.md) を参照。

**処理内容:**

1. ユーザ削除情報メッセージ送信テーブルから送信対象データを抽出し、ユーザ削除情報電文として外部システムへ送信
   - INPUT: ユーザ削除情報メッセージ送信テーブル
   - OUTPUT: ユーザ削除情報電文
