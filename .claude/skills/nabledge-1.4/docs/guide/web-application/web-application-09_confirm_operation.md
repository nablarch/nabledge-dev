# 動作確認の実施

## 動作確認の実施

ユーザ情報登録機能の動作確認手順:

1. ユーザ情報登録画面を表示する。
2. 入力フォームの値を変更して **確認** ボタンを押下する。
3. ユーザ情報登録確認画面に遷移するので、登録する値が表示されていることを確認し、 **確定** ボタンを押下する。
4. ユーザ情報登録完了画面に遷移するので、登録した値が表示されていることを確認する。
5. USERSテーブルを確認し、登録したユーザ情報が格納されていることを確認する。

> **注意**: ユーザ情報登録確認画面のキャンセルボタンは未実装のため動作しない。
