# disabled 項目切替えイベントアクション

## disabled 項目切替えイベントアクション

チェックボックス・ラジオボタン・プルダウン要素の選択状態に応じて、特定の要素の `disabled` 属性値を動的に切替えるイベントアクション。

仕様は [event_toggle_readonly](ui-framework-event_toggle_readonly.md) と同等（切替え対象の属性が `disabled` である点を除く）。
