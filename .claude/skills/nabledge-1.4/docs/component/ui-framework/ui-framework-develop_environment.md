## 統合開発環境の補完機能を利用する

※記載してあるショートカットは、標準から変更していない場合のショートカット。

タグを途中まで入力して `C-SPC` を押下すると、入力した文字にマッチする利用可能なタグの候補が表示される。

[eclipse-template](ui-framework-template_list.md) を導入することで、テンプレートを利用した補完も行えるようになる。テンプレートを選択すると登録済みのテンプレートが展開され、入力項目間は `TAB` キーで遷移できる。
