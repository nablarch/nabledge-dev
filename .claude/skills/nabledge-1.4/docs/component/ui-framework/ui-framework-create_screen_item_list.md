# 業務画面JSPから画面項目定義を作成する

## 業務画面JSPから画面項目定義を作成する

業務画面JSPをブラウザで開き右クリック →「設計書ビューを開く」を選択すると、画面項目一覧が別ウィンドウ（タブ）に表示される。セルはCtrlキー＋ドラッグで選択し、Ctrl+Cでコピーして設計書のExcelシートに直接貼り付け可能。

> **補足**: 画面項目一覧に表示されるのは業務画面JSPに**直接記述**した項目のみ。`<jsp:include>`等を経由して描画する項目はJSPのプレビューには表示されるが、画面項目一覧には表示されない。これは共通化された表示項目は共通部品として個別に設計され、各画面の設計書から参照される形となるため。

> **補足**: 図はNablarch標準のシステム機能設計書様式に基づく。プロジェクトが標準様式を修正している場合は表示が異なる。
