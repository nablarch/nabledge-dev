# disabled 項目切替えイベントアクション

[disabled 項目切替えイベントアクション](../../component/ui-framework/ui-framework-event-toggle-disabled.md) は、
チェックボックス・ラジオボタン・プルダウン要素の選択状態に応じて、
特定の要素の disabled 属性値を動的に切替えるイベントアクションである。

本イベントアクションの仕様は、切替え対象の属性が disabeld となっている点を除けば
[readonly 項目切替えイベントアクション](../../component/ui-framework/ui-framework-event-toggle-readonly.md) と同等である。

詳細はそちらのドキュメントを参照すること。
