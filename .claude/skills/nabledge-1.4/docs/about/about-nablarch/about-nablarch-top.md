# Nablarch 1.4.11版

## Nablarchについて

Nablarch(ナブラーク)は、TISの豊富な基幹システム構築経験から得られたナレッジを集約したJavaアプリケーション開発/実行基盤です。

<summary>keywords</summary>

Nablarch, ナブラーク, Javaアプリケーション開発実行基盤, TIS, 基幹システム構築

</details>

## リファレンス

**アプリケーション実行環境**
- アプリケーションフレームワーク: オブジェクトコード、依存ライブラリ、ソースコード、解説書、APIドキュメント（アーキテクト向け・アプリケーションプログラマ向け）

**アプリケーション開発環境**
- アプリケーション開発基盤: 開発環境構築ガイド、開発リポジトリ構築ガイド
- UI開発基盤: 解説書、JSP/HTML作成ガイド
- モバイルライブラリ: 解説書、iOS版
- ワークフローライブラリ: 解説書、設計ガイド、定義データ自動生成ツール、実装ガイド、サンプルアプリケーション
- アプリケーションライブラリ: 業務機能サンプル、拡張モジュールサンプル、メッセージング基盤テストシミュレータサンプル、運用機能サンプル
- テスティングフレームワーク: オブジェクトコード、依存ライブラリ、APIドキュメント（アーキテクト向け・アプリケーションプログラマ向け）
- ツールボックス: 解説書

**アプリケーション開発標準**
- 開発プロセス標準、設計標準、コーディング規約、単体テスト標準、ドキュメント規約、設計書

**アプリケーション開発ガイド**
- プログラミング・単体テストガイド
