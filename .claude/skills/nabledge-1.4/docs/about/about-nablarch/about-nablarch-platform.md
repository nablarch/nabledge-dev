## Nablarchフレームワークの稼動環境

Nablarchフレームワーク以外のコンテンツ(例えばUI開発基盤など)に関する稼動環境については、各コンテンツのドキュメントを参照してください。

### Nablarchフレームワークの環境要件

* J2SE 5.0/ Java SE 6/ Java SE 7/ Java SE 8
* JDBC 3.0
* JavaBeans(TM) Activation Framework 1.1
* Java Servlet 2.5
* Java API for JavaServer Pages 2.1
* JavaServer Pages Standard Tag Library 1.1
* Java EE JMS API 1.1-rev-1
* JavaMail API 1.4

### Nablarchフレームワークの稼動確認済み環境

Nablarchフレームワークは、下記の環境においてテストを実施し、正常に動作することを確認しています。

#### アプリケーションサーバ

* Oracle Weblogic Server 11g/12c
* WebSphere Application Server 7
* JBoss Application Server 7
* Apache Tomcat 6/7

#### データベースサーバ

* Oracle Database 11g
* DB2 9
* SQLServer 2008

#### MOM（メッセージ指向ミドルウェア）

* WebSphere MQ 7

#### ブラウザ

* Internet Explorer 6/7/8/9/10/11
* Mozilla Firefox 3.6
* Google Chrome 10
