## Nablarchの知的財産権と利用制限

Nablarchの知的財産権と利用制限について説明します。

### Nablarchの知的財産権

Nablarchの知的財産権は、TISに帰属します。 ただし、お客様が改変・翻案した部分の知的財産権は、お客様に帰属します。

### Nablarchの利用制限

お客様はNablarchのコンテンツを使用、複製、改変・翻案することができます。ただしコンテンツ種類により、以下の制限があります。

Nablarchコンテンツ種類（Nablarchフレームワーク、 Nablarch標準/ガイド、 Nablarchライブラリ、Nablarchツール）については、 [Nablarchのコンテンツ種類](../../about/about-nablarch/about-nablarch-contents-type.md#contents-type) を参照。

| コンテンツ種類 | 利用上の制限 | 備考 |
|---|---|---|
| Nablarchフレームワーク | * 改変・翻案部はサポートサービスの対象外となります。 * Nablarchフレームワークに依存せず単独で動作するように   改変・翻案することはできません。 | Nablarchフレームワークの改変・翻案は、 推奨いたしません。 [1] |
| Nablarch標準/ガイド | * 改変・翻案部はサポートサービスの対象外となります。 * Nablarchフレームワークと組み合わせて   利用していただく必要があります。 |  |
| Nablarchライブラリ |  |  |
| Nablarchツール |  |  |

Nablarchフレームワーク（アプリケーションフレームワークおよびテスティングフレームワーク）の更新バージョンは、後方互換性が維持されます。
お客様がNablarchフレームワークを改変・翻案した場合、バージョンアップ時に後方互換が維持されず、思わぬ不具合が発生する可能性があります。
Nablarchの後方互換性については、 [後方互換性ポリシー](../../about/about-nablarch/about-nablarch-versionup-policy.md#backward-compatibility-policy) を参照。
