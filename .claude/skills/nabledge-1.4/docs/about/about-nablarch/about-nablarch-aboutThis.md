## 注意事項

## 注意事項

サンプルプログラムのソースコードに関する注意事項:

- 説明箇所を明示するため一部内容を省略している（～中略～と表記）。ソースコードの全量はNablarchのサンプルアプリケーションを参照すること。
- 見やすさのため改行位置を変えている箇所がある。
- 以下の書き出しで始まるコメントは説明のために追加したものであり、実際のソースコード作成時には記述**不要**:
  - `` // 【説明】 ``
  - `` /* 【説明】 ``
  - `` <!-- 【説明】 ``
  - `` <%-- 【説明】 ``
  - `` -- 【説明】 ``

<details>
<summary>keywords</summary>

サンプルコード注意事項, コメント規則, ソースコード省略, 説明コメント

</details>
