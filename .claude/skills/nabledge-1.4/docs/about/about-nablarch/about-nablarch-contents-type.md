## Nablarchのコンテンツ種類

Nablarchのコンテンツ種類について説明します。

| コンテンツ種類 | 説明 | 含まれるコンテンツ |
|---|---|---|
| Nablarchフレームワーク | アプリケーション実行基盤です。 オブジェクトコードおよびソースコードが含まれます。 | アプリケーションフレームワーク テスティングフレームワーク |
| Nablarch標準/ガイド | アプリケーション開発で必要な開発標準およびガイドです。 必要に応じて、お客様のPJ要件に合わせたカスタマイズ を行う必要があります。 | アプリケーション開発基盤 アプリケーション開発標準 アプリケーション開発ガイド アプリケーションフレームワーク解説書 テスティングフレームワーク解説書 |
| Nablarchライブラリ | お客様のPJ要件に合わせたカスタマイズを前提した ソースコードライブラリです。 | アプリケーションライブラリ UI開発基盤 モバイルライブラリ |
| Nablarchツール | アプリケーション開発の品質と生産性を向上するツール群です。 必要に応じて、お客様のPJ要件に合わせたカスタマイズ を行う必要があります。 | ツールボックス |
