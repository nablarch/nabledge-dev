## コード機能のpattern属性の指定方法

**Q**: コード名称やコード値の存在チェック機能で `pattern` 属性に「2」などの数値を指定するとエラーになる。

**A**: `pattern` 属性には、コードテーブルの対応するカラムの**物理名**を設定する必要がある。

例: パターン2のカラム物理名が `PATTERN2` の場合 → `pattern` 属性には `PATTERN2` と指定する。
