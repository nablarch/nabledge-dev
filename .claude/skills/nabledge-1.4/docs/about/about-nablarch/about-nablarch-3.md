# コード名称やコード存在チェック時に指定するパターンには、何を指定したらいいのでしょうか？

> **question:**
> コード機能で、コード名称やコード値の存在チェックを行う機能でパターンを指定するものがあります。
> このパターンには、何を指定したらいいのでしょうか？

> 例えば、パターン2を使用したいので、pattern属性に「2」を指定して見ましたが、パターンが存在しないとのエラーが発生してしまいます。

> **answer:**
> pattern 属性には、コードテーブルの対応するカラムの物理名を設定する必要があります。

> 例えば、パターン2のカラム物理名がPATTERN2の場合には、 pattern 属性にはPATTERN2と指定する必要があります。
