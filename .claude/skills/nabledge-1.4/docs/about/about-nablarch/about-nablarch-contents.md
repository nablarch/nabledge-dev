## Nablarchの製品構成

Nablarchのコンテンツについて説明します。

| コンテンツ名 |  | 説明 |
|---|---|---|
| アプリケーション 実行環境 | アプリケーション フレームワーク | Java実行環境上で実行可能なアプリケーション実行基盤です。 オブジェクトコードおよびソースコードが含まれます。 |
|  | アプリケーション フレームワーク解説書 | アプリケーションフレームワークの解説書です。 |
| アプリケーション 開発環境 | アプリケーション 開発基盤 | Nablarchを使用してアプリケーション開発を行う際の開発環境およびガイドです。 |
|  | UI開発基盤 | 画面開発のQCDを要件定義からPG/UTまで一体で向上させる開発基盤です。 |
|  | モバイルライブラリ | iOS、androidといったモバイルのネイティブアプリケーション開発に使用するライブラリです。 |
|  | アプリケーション ライブラリ | 実プロジェクトで頻繁に作成する機能のサンプルです。 業務機能サンプル、Nablarch拡張モジュールサンプル、運用機能サンプル等が含まれます。 |
|  | テスティング フレームワーク | 自動テストフレームワークや静的解析ツール、テスト支援ツールを含むテスティングフレームワークです。 オブジェクトコードとソースコードが含まれます。 |
|  | テスティング フレームワーク解説書 | テスティングフレームワークの解説書です（Javadoc）。 |
|  | ツールボックス | プログラムやテストデータの自動生成ツールなど、アプリケーション開発の品質 と生産性を向上するツール群です。 |
| アプリケーション 開発標準/ガイド | アプリケーション 開発標準 | 開発プロセス標準、DB設計標準、UI標準、コーディング規約、単体テスト標準等 のアプリケーション開発に必要な標準文書 および 設計書フォーマットです。 |
|  | アプリケーション 開発ガイド | Nablarchフレームワークを使用してアプリケーションの実装とテストを実施する方法 を説明したドキュメントです。 初期学習用チュートリアルも含まれます。 |

各コンテンツについて、ソースコードの開示・非開示、特定のOSSやミドルウェアへの依存の有無を説明します。

| コンテンツ名 |  | ソースコード開示 [1] | OSSの利用 [2] | 特定M/Wへの依存 [3] |
|---|---|---|---|---|
| アプリケーション 実行環境 | アプリケーション フレームワーク | 開示 | なし | なし |
|  | アプリケーション フレームワーク解説書 | － | － | － |
| アプリケーション 開発環境 | アプリケーション 開発基盤 | － | あり | あり |
|  | UI開発基盤 | 開示 | あり | なし |
|  | モバイルライブラリ | 開示 | あり | あり |
|  | アプリケーション ライブラリ | 開示 | あり | あり |
|  | テスティング フレームワーク | 開示 | あり | なし |
|  | テスティング フレームワーク解説書 | － | － | － |
|  | ツールボックス | 開示 | あり | あり |
| アプリケーション 開発標準/ガイド | アプリケーション 開発標準 | － | － | － |
|  | アプリケーション 開発ガイド | － | － | － |

ソフトウェアの場合に、ソースコードを開示するかどうか。ソースコード開示のコンテンツについては、ソースコードを納品します。

オープンソースソフトウェアを使用している、または今後使用する可能性があるか否か。
ドキュメントの場合は、オープンソースソフトウェアを使用することを前提とした記述がある、または今後記述する可能性があるか否か。
本番環境のサーバサイドで稼働させるソフトウェアについては、オープンソースソフトウェアを使用しない方針としています。

特定のミドルウェアに依存している、または今後依存する可能性があるか否か。
ドキュメントの場合は、特定のミドルウェアを使用することを前提とした記述がある、または今後記述する可能性があるか否か。
