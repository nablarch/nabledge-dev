# Excelのテストデータに設定した整数値が、テスト実行時には小数部ありの数値となってしまいます。回避方法を教えてください

## Excelテストデータの整数値が小数に変換される問題と回避方法

ExcelテストデータのAGE列などに整数値（例: `20`）を設定すると、テスト実行時に`20.0`のように小数部ありの数値に変換される問題がある。

> **重要**: Excelに記述するテストデータの書式は全て**文字列書式**で設定すること。標準や数値書式など文字列書式以外の場合、自動テストFWがExcelに設定されたデータを正しく読むことができない。
