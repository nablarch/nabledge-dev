# Shell Script自動生成ツール

## 概要

Shellスクリプト及びShell実行用の環境依存値を持つ設定ファイルを自動生成するツール。

入力ファイル:
- :download:`シェル共通設定<../tools/ShellGenerator/シェル共通設定.xls>`
- :download:`ジョブ実行シェルスクリプト自動生成設定<../tools/ShellGenerator/ジョブ実行シェルスクリプト自動生成設定.xls>`

自動生成されるShellスクリプトの対応処理:
- 都度起動バッチ処理
- 常駐バッチ処理
- メッセージング
- ファイル転送処理
