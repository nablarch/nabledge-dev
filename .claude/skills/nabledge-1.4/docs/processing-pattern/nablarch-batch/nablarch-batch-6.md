# データベースをインプットとするバッチ処理でも、画面処理と同じようにFormは必要ですか？

> **question:**
> データベースをインプットとするバッチ処理でも、画面処理と同じようにFormを作成する必要があるのでしょうか？
> それとも、必要な時に入力データをEntityに変換して処理するほうがいいのでしょうか？

> **answer:**
> 入力データを単一のテーブルに出力するような場合には、Formを作らずEntityのみで処理する方が良いでしょう。

> ただし、複数テーブルに出力するような場合には、Formを作成して処理対象データをFormデータで引き回したほうが良いでしょう。
