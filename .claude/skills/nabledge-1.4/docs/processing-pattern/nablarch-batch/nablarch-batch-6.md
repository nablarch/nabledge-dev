# データベースをインプットとするバッチ処理でも、画面処理と同じようにFormは必要ですか？

## バッチ処理でのForm要否判断

データベースをインプットとするバッチ処理において、Formの要否は出力先テーブル数で判断する。

- **単一テーブルへの出力**: Formを作らずEntityのみで処理する。
- **複数テーブルへの出力**: Formを作成し、処理対象データをFormデータで引き回す。
