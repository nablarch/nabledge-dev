# 画面に表示するエラーメッセージを途中で改行することは出来ますか?

> **question:**
> 画面に表示するエラーメッセージ等のメッセージを途中で改行したいのですが、改行させる方法が分かりません。

> メッセージに *<br />* を含めて見ましたが、サニタイジングされるため *<br />* が画面表示されてしまいます。

> **answer:**
> 画面に表示するメッセージに物理的な改行を含めるようにしてください。
> これにより、taglibでは、表示対象のデータに改行が含まれていると、<br />タグに置き換えて出力を行ないます。

> 例えば、データベースでメッセージを管理する場合、データベースには以下のようにデータを登録する事により、
> 画面表示時に改行させることができるようになります。

> ```sql
> -- CHR(13) || CHR(10)は、「\r\n」を表す
> INSERT INTO MESSAGE (MESSAGE_ID, MESSAGE_TEXT) VALUES ('ID01', 'line 1' || CHR(13) || CHR(10) || 'line2');
> ```
