# 画面に表示するエラーメッセージを途中で改行することは出来ますか?

## 画面に表示するエラーメッセージを途中で改行することは出来ますか?

画面表示メッセージを改行するには、メッセージに物理的な改行（`\r\n`）を含めること。taglibは表示対象データに改行が含まれている場合、`<br />`タグに置き換えて出力する。メッセージテキストに`<br />`を直接含めるとサニタイジングされそのまま表示されるため、この方法は使用不可。

データベースでメッセージを管理する場合の登録例:

```sql
-- CHR(13) || CHR(10)は、「\r\n」を表す
INSERT INTO MESSAGE (MESSAGE_ID, MESSAGE_TEXT) VALUES ('ID01', 'line 1' || CHR(13) || CHR(10) || 'line2');
```
