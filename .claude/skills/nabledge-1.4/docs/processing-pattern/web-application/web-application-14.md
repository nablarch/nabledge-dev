## Firefoxで2重サブミット防止が効かない問題

Firefoxの古いバージョン（3.x系を含む）で、`n:submitLink`で生成されたリンクを素早くダブルクリックすると2回連続でリクエストが送信されることがある。Chrome/Safari/IE6では発生しない。

Firefox 9.01以降にアップデートすることで解消する。

> **重要**: 二重サブミット防止機能はサーバサイドのトークンチェックと併用することが前提。本事象が発生しても、業務処理が2重実行されることはない。
