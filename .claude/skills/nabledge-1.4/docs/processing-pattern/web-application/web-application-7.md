# コード値精査時に指定するパターンには何を指定するのでしょうか？

> **question:**
> コード値精査( *CodeValue* )でパターンを指定して精査を実施したいのですが、パターンには何を指定したらいいのでしょうか?

> パターン2を使用して精査を行いたいので、 *pattern* 属性には **2** を指定してみましたがパターンが存在しないとのエラーが発生してしまいます。
