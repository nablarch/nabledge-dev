# 業務画面JSPから画面項目定義を作成する

**公式ドキュメント**: [業務画面JSPから画面項目定義を作成する](https://nablarch.github.io/docs/LATEST/doc/development_tools/ui_dev/guide/widget_usage/create_screen_item_list.html)

## 業務画面JSPから画面項目定義を作成する

業務画面JSPをブラウザで開き、画面上で右クリック → メニュー最上段「設計書ビューを開く」を選択 → 設計書ビューが別ウィンドウ(タブ)に表示される。

設計書ビューのセル選択: **Ctrl** キーを押しながらマウスでドラッグ（セル単位で選択可能）。選択セルは **Ctrl+C** でコピーし、システム機能設計書のExcelシートに直接貼り付け可能。

> **重要**: 設計書ビューの左下に表示される「画面設計ボタン」はクリックしないこと。誤ってクリックした場合は「設計書ビューを構築しています…」の状態で画面が固まる。その場合は設計書ビューウィンドウをクローズし、再度「設計書ビューを開く」から設計書ビューを開くこと。

> **補足**: 画面項目一覧に表示される項目は、業務画面JSPに直接記述した項目のみ。`<jsp:include>` などを経由して描画する項目はJSPのプレビューには表示されるが、画面項目一覧には表示されない（共通化された表示項目は共通部品として個別に設計され、各画面の設計書から参照される形となるため）。

> **補足**: 表示される図はNablarch標準のシステム機能設計書の様式に沿ったもの。プロジェクト側で標準様式を修正している場合は、図とは異なる表示になる。
