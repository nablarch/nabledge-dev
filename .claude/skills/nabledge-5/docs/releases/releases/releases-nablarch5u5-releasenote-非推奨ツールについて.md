# 非推奨ツールについて

非推奨ツールについて
以下のツールの新規利用は非推奨といたします。

|  |  | No | ツール | ツールの概要 | 提供形態 | 非推奨の理由 | 代替手段 |
|---|---|---|---|---|---|---|---|
|  |  | 1 | Form自動生成ツール | データベースアクセス用エンティティ、画面入力に使用するForm、外部インターフェースに使用するFormを自動生成する。 | nablarch-toolbox (Mavenリポジトリからjarファイルで提供) | 以下の理由により、外部インターフェース用のFormの自動生成を除き、Nablarch5の推奨方式と異なるため。 ・データベースアクセス用のエンティティはGSPプラグインから生成する方法を推奨している。 ・画面入力に使用するFormはデータベース設計書からではなく、画面の入力項目に従って手作業にて作成することを推奨している。 | 本ツールは外部インターフェース用のForm生成のみに使用してください。 データベースアクセス用のエンティティの生成はGSPプラグインを使用してください。 画面入力項目に使用するFormは手作業にて作成してください。 |
|  |  | 2 | JSP自動生成ツール | HTMLファイルを基にJSPファイルを自動生成する。 | nablarch-toolbox (Mavenリポジトリからjarファイルで提供) | デザインHTML内のタグを全てnタグに置換するとデザインが崩れてしまい、有効に活用できるケースが極めて限定されるため。 | HTMLからJSPへの変換は手作業にて必要な個所を変換してください。 |
|  |  | 3 | SQL自動生成ツール | テーブル定義書を基にSQL文を自動生成する。 | Excelマクロ | 本ツールの機能はUniversalDaoを利用することで、利用する必要が無くなるため。 | UniversalDaoを使用してください。 |
|  |  | 4 | 環境設定ファイル自動生成ツール | 環境設定ファイルを自動生成する。 | nablarch-toolbox (Mavenリポジトリからjarファイルで提供) | 以下の理由により、利便性の面で問題があるため。 ・ビルド時にExcelからconfigファイルが生成されるという挙動が、多くの開発者にとって予想外であった。 ・Excelファイルは差分比較を行うのに手間が掛かるため、変更管理の面で不便であった。 | configファイルを直接テキストエディタなどで編集するようにしてください。 |
|  |  | ※上記ツールを既にお使いの場合は継続利用いただけます。jarファイルで提供しているツールについては、ツールのjarファイルをバージョンアップした後も利用いただけます。 |  |  |  |  |  |
