# ご利用にあたって

**公式ドキュメント**: [ご利用にあたって](https://nablarch.github.io/docs/LATEST/doc/terms_of_use/index.html)

## 情報の利用目的

取得した情報はアクセス解析に利用される。ページのアクセス数等からサイトの利用状況を把握し、サイトの改善に利用する。アクセス情報は匿名で収集され、個人を特定することはできない。

<details>
<summary>keywords</summary>

アクセス解析, プライバシー, 利用目的, 匿名収集, サイト改善

</details>

## 情報の送信先

取得した情報は [Plausible Analytics](https://plausible.io) に送信される。

<details>
<summary>keywords</summary>

Plausible Analytics, 情報送信先, 外部送信

</details>

## 情報の種類・用途

| 情報の種類 | 例 | 説明 | 用途 |
|---|---|---|---|
| ページのURL | https://nablarch.github.io/docs/LATEST/doc/index.html | 当ウェブサイトのURL | どのページが閲覧されたか測定するために使用 |
| リファラー | https://x.com/ | 参照元ページ | どのページから到達したかを測定するために使用 |
| ブラウザ | Chrome 86.0 | 利用者のブラウザの種類 | 利用者の閲覧環境を測定するために使用 [1] |
| OS | macOS 10.15 | 利用者のOSの種類 | 利用者の閲覧環境を測定するために使用 [2] |
| デバイス種別 | Desktop | 利用者のデバイスの種類 | 利用者の閲覧環境を測定するために使用 [3] |
| 国、地域、都市 | United Kingdom, England, London | 利用者の位置情報 | 利用者の閲覧環境を測定するために使用 [4] |

[1] ユーザーエージェントから取得するが、必要な情報だけを使用。完全なユーザーエージェントは破棄される。

[2] ユーザーエージェントから取得するが、必要な情報だけを使用。完全なユーザーエージェントは破棄される。

[3] ユーザーエージェントから取得するが、必要な情報だけを使用。完全なユーザーエージェントは破棄される。

[4] IPアドレスを使用して位置情報を取得するが、都市レベルより詳細な追跡は行わない。IPアドレスは破棄され、データベース等に保存されない。

詳細は [Plausible Analyticsのポリシー](https://plausible.io/data-policy) を参照。
