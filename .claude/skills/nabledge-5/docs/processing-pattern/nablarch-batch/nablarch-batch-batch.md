## バッチフレームワーク選択

2種類のバッチフレームワークを提供:
- [jsr352/index](../jakarta-batch/jakarta-batch-jsr352.md)
- [nablarch_batch/index](nablarch-batch-nablarch_batch.md)

[nablarch_batch/index](nablarch-batch-nablarch_batch.md) の使用を推奨する。理由: JSR352は2020年時点で情報が少なく有識者もアサインしにくい。

> **補足**: Nablarch5u15までの解説書ではJSR352準拠バッチアプリケーションを推奨していたが、2020年現在の普及状況と学習コストの高さを鑑み、Nablarchバッチアプリケーションを推奨する方針に転換した。
