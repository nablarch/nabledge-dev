# オンラインアクセスログ集計機能

## 概要

オンラインアクセスログからリクエストID単位で以下を集計:
- リクエスト数
- 閾値超過リクエスト数
- 処理時間（平均）
- 処理時間（中央値）
- 処理時間（最大値）

> **補足**: 設定ファイルでリクエストIDを指定し、集計対象機能を絞り込み可能。

## サンプル構成

本サンプルは3種類で構成:

| サンプル名 | 概要 |
|---|---|
| オンラインアクセスログ解析バッチ | アクセスログを解析し、集計用情報をCSVに出力 |
| オンラインアクセスログ解析結果集計バッチ | 解析結果CSVを集計。集計期間は設定ファイルで指定 |
| オンラインアクセスログ集計結果レポートサンプル | 集計結果からExcelレポートを生成するマクロ |

### 処理の流れ

1. オンラインログ配置サーバで解析バッチ実行
2. 解析結果CSVを集計バッチで処理
3. 集計結果CSVからExcelマクロでレポート生成

> **補足**: アクセスログには個人情報が含まれる可能性があるため、セキュリティ保護環境での実行を推奨。集計結果は個人情報を含まないため、保護環境外でも実行可能。

## オンラインアクセスログ解析バッチ

アクセスログを解析し、集計用情報のみをCSVに出力するバッチ処理。

**実行想定**: 日次実行

> **重要**: 解析結果CSVは削除せず蓄積すること。過去分CSVがないと集計バッチで正確な集計ができない。

**出力ファイル名**: `REQUEST_INFO_{システム日付8桁}.csv`

**CSV出力項目**:

| 項目名 | 備考 |
|---|---|
| 年 | リクエスト終了(END)ログ出力日時の年 |
| 月 | リクエスト終了(END)ログ出力日時の月 |
| 日 | リクエスト終了(END)ログ出力日時の日 |
| プロセス名 | ログにプロセス名が出力されていない場合はブランク |
| リクエストID | リクエストID |
| 処理時間 | リクエストの処理時間 |
| ステータスコード | 処理ステータスコード |

### 設定情報

`オンラインアクセスログ解析及び集計サンプルの設定`_を参照。

### 実行方法

:ref:`Nablarchのバッチ方式<nablarch_batch>`を使用。

**必須パラメータ**:

| パラメータ | 値 |
|---|---|
| diConfig | `statistics-batch.xml` (resourcesをクラスパス設定時) |
| requestPath | `OnlineAccessLogParseAction` (バッチアクションクラス名) |
| userId | バッチユーザID |

## オンラインアクセスログ解析結果集計バッチ

解析結果CSVを集計するバッチ処理。集計期間は設定ファイルで指定。

> **補足**: 対象日数判定はファイル名の日付で行う。解析が日次実行されていない場合、集計範囲より前のログも含まれることがある。例: 集計期間2日でも、解析が2日に1回実行の場合、3日分が集計対象になる可能性あり。

**出力ファイル**: 3種類のCSV

| ファイル種別 | 出力内容 |
|---|---|
| 時間別集計結果 | 時間単位の集計 |
| 日別集計結果 | 日単位の集計 |
| 年月別集計結果 | 年月単位の集計（システム日付の年月のみ） |

> **重要**: 年月別集計結果は過去分を削除せず蓄積すること。

> **補足**: 集計範囲が1ヶ月未満の場合、年月集計結果は指定日数分のみの集計となる。例: 30日実行で集計範囲10日の場合、20～30日が対象。

**CSV出力項目**:

| 項目名 | 備考 |
|---|---|
| リクエストID | リクエストID |
| 集計対象期間 | 時間別:0～23、日別:1-31、年月別:システム日付の年月 |
| プロセス名 | プロセス名 |
| リクエスト数 | 集計対象期間内のリクエスト数 |
| 処理時間が閾値を超えたリクエスト数 | 設定ファイルで指定の閾値超過数 |
| 処理時間（平均） | 集計対象期間内の平均値 |
| 処理時間（中央値） | 集計対象期間内の中央値 |
| 処理時間（最大値） | 集計対象期間内の最大値 |

### 設定情報

`オンラインアクセスログ解析及び集計サンプルの設定`_を参照。

### 実行方法

:ref:`Nablarchのバッチ方式<nablarch_batch>`を使用。

**必須パラメータ**:

| パラメータ | 値 |
|---|---|
| diConfig | `statistics-batch.xml` (resourcesをクラスパス設定時) |
| requestPath | `RequestInfoAggregateAction` (バッチアクションクラス名) |
| userId | バッチユーザID |

## オンラインアクセスログ集計結果レポートサンプル

集計結果からExcelレポート（集計結果表）を生成するマクロ。グラフ作成はExcel機能を使用。

### 実行方法

使用方法詳細: `/tool/ウェブアプリケーションリクエストレポートツール.xls`を参照。

## オンラインアクセスログ解析及び集計サンプルの設定

解析・集計バッチの設定。

**設定クラス**: `please.change.me.statistics.action.settings.OnlineStatisticsDefinition`

**設定ファイル**:
- `main/resources/statistics/onlineStatisticsDefinition.xml`
- `main/resources/statistics/statistics.config`

全プロパティ必須。標準設定が用意されているため、環境に応じて必要な項目のみ変更。

### 設定プロパティ

**ログ解析関連**:

| プロパティ名 | 設定内容 |
|---|---|
| accessLogDir | 解析対象ログのディレクトリパス（絶対/相対） |
| accessLogFileNamePattern | ログファイル名パターン（`*`でワイルドカード、正規表現ではない） |
| accessLogParseDir | 解析用一時ディレクトリパス（ログをコピーして解析） |
| endLogPattern | 終了ログ特定用正規表現 |
| includeRequestIdList | 解析対象リクエストIDリスト |

> **補足**: リクエストID増減時は`includeRequestIdList`を更新すること。

**ログ抽出パターン**:

| プロパティ名 | 設定内容 |
|---|---|
| findRequestIdPattern | リクエストID抽出用正規表現（抽出部分をグループ化） |
| findProcessNamePattern | プロセス名抽出用正規表現（抽出部分をグループ化） |
| findStatusCodePattern | ステータスコード抽出用正規表現（抽出部分をグループ化） |
| logOutputDateTimeStartPosition | ログ出力日時の開始位置（0始まり、`String#substring`仕様） |
| logOutputDateTimeEndPosition | ログ出力日時の終了位置（0始まり） |
| logOutputDateTimeFormat | ログ出力日時フォーマット（`SimpleDateFormat`形式） |
| findExecutionTimePattern | 処理時間抽出用正規表現（抽出部分をグループ化） |

**集計設定**:

| プロパティ名 | 設定内容 |
|---|---|
| thresholdExecutionTime | 処理時間閾値（ミリ秒）。例: 3000で3秒超過リクエスト数を集計 |
| aggregatePeriod | 集計期間（日数）。年月集計を正確にするため最低30を推奨 |

**ファイル設定**:

| プロパティ名 | 設定内容 |
|---|---|
| requestInfoFormatName | 解析結果CSVのフォーマット定義ファイル名（デフォルト: `main/format/requestInfo.fmt`） |
| requestInfo.dir | 解析結果CSV出力先ディレクトリの論理名（マッピング: `main/resources/statistics/file.xml`） |
| requestInfoSummaryBaseName | 集計結果CSV出力先ディレクトリの論理名（マッピング: `main/resources/statistics/file.xml`） |
| requestInfoSummaryFormatName | 集計結果CSVのフォーマット定義ファイル名（デフォルト: `main/format/requestInfoAggregate.fmt`） |

> **補足**: フォーマット定義ファイルは通常デフォルトのまま。バッチ拡張で項目を追加/削除した場合のみ、対応するフォーマット定義ファイルを作成し設定。
