## 概要

> **補足**: Nablarch5まで「JSP静的解析ツール」という名称だったが、Java EEがEclipse Foundationに移管されて仕様名が変わったため「Jakarta Server Pages静的解析ツール」に名称変更された。機能的な差は無く、名称のみの変更。Nablarch6で名称変更された他の機能については :ref:`renamed_features_in_nablarch_6` を参照。
