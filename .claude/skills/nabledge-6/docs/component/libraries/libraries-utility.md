# 汎用ユーティリティ

## ユーティリティクラス一覧

| クラス名 | モジュール | 説明 |
|---|---|---|
| `DateUtil` | nablarch-core | 日付に関する機能 |
| `FileUtil` | nablarch-core | ファイルの取り扱いに関する機能 |
| `ObjectUtil` | nablarch-core | オブジェクトの取り扱いに関する機能 |
| `StringUtil` | nablarch-core | 文字列に関する機能 |
| `BeanUtil` | nablarch-core-beans | Java Beansクラスに関する機能。詳細は :ref:`bean_util` 参照 |
| `Base64Util` | nablarch-core | Base64エンコーディング機能。Java 8以降は `Base64エンコーディングの標準API` が提供されており、本ユーティリティは後方互換のために存在 |
| `BinaryUtil` | nablarch-fw-web-extension | バイナリに関する機能 |
