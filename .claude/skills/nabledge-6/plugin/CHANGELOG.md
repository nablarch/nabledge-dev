# Changelog

All notable changes to the nabledge-6 plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.2] - 2026-02-13

### Changed
- セットアップスクリプトがjqを自動インストール（Linux/WSL/GitBash）
- READMEから手動インストール手順を削除し、自動インストールについて記載

## [0.1] - 2026-02-13

### Added
- Nablarch 6u3の知識検索機能
- コード分析機能（構造化テンプレート）
- バッチ処理の基礎知識
- データベースアクセスライブラリ
- テスティングフレームワークの基礎
- セキュリティチェックリスト
- Claude Code用セットアップスクリプト（setup-6-cc.sh）
- GitHub Copilot用セットアップスクリプト（setup-6-ghc.sh）
