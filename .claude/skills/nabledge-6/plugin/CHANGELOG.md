All notable changes to the nabledge-6 plugin will be documented in this file.

## [0.4] - 2026-02-13

### Fixed
- setup.shを修正：配布リポジトリ構造から利用者向けskill構造に正しく展開

- 不要な.claude-plugin/をコピーしないように修正

## [0.3] - 2026-02-13

### Fixed

## [0.2] - 2026-02-13

### Fixed

- テスティングフレームワークの基礎
- セキュリティチェックリスト
