# 変更履歴

nabledge-1.2プラグインの主な変更内容を記録しています。

フォーマットは [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) に基づいています。

## [Unreleased]

### 変更
- 知識ファイルの生成方式をAIによる要約からルールベース変換に切り替えました。AI特有のハルシネーションが発生しなくなり、ミッションクリティカルなシステムでも安心して利用できるようになりました。

### 修正

- コード分析のシーケンス図で、HTTPパス形式がJavaメソッド名に正しく変換されない問題を修正しました
- コード分析の処理フローで、ヘルパー/プライベートメソッドが省略される問題を修正しました
- コード分析の出力ファイルにファイルパス情報を追加しました

## [0.1] - 2026-03-30

### 追加

- Nablarch 1.2の全ドキュメントをカバーする知識ファイルを追加しました。
