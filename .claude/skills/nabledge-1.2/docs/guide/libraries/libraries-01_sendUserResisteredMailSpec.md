# サンプルアプリケーションの概要

## サンプルアプリケーションの概要

ユーザ情報登録機能では、登録ユーザーのメールアドレス宛に以下の定型メールを送信する。

| 件名 | 本文 |
|---|---|
| {漢字氏名} さんのユーザー登録が完了しました。 | {漢字氏名} さん<br><br>Nablarch Sampleアプリケーションへのユーザー登録が完了しました。<br><br>ログインID：{ログインID}<br><br>下記URLからログインしてください。<br><br>http://localhost:8999/action/ss11AB/W11AB01Action/RW11AB0101 |

`{漢字氏名}`、`{ログインID}` には画面で入力した値が挿入される。
