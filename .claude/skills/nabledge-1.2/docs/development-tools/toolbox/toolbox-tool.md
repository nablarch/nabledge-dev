## Nablarch Toolbox

本書では、アプリケーション開発時に使える便利なツールを紹介する。
Nablarch Toolbox には試作段階のツールが含まれる場合がある。

| ツール | 概要 |
|---|---|
| [Form自動生成ツール](../../development-tools/toolbox/toolbox-01-EntityGenerator.md#form自動生成ツール) | Formを自動生成する。 |
| [JSP自動生成ツール](../../development-tools/toolbox/toolbox-01-JspGenerator.md#jsp自動生成ツール) | HTMLファイルを基にJSPファイルを自動生成する。 |
| [環境設定ファイル自動生成ツール](../../development-tools/toolbox/toolbox-ConfigGenerator.md#環境設定ファイル自動生成ツール) | 環境設定ファイルを自動生成する。 |
| [SQL自動生成ツール](../../../knowledge/assets/toolbox-tool/SQL自動生成ツール.xls) | テーブル定義書を基にSQL文を自動生成する。 |
| [排他制御主キークラス自動生成ツール](../../../knowledge/assets/toolbox-tool/排他制御主キークラス自動生成ツール.xls) | テーブル定義書を基に排他制御主キークラスを自動生成する。なお、本ツールを使用するには、 [物理型・Javaオブジェクトマッピング定義](../../../knowledge/assets/toolbox-tool/物理型・Javaオブジェクトマッピング定義.xls) が必要。 |
| [フォーマット定義ファイル自動生成ツール](../../../knowledge/assets/toolbox-tool/フォーマット定義ファイル自動生成ツール.xls) | 外部インタフェース設計書から、FWが使用するフォーマット定義ファイルを自動生成する。 |
| [リクエストテーブル登録用データ作成ツール](../../../knowledge/assets/toolbox-tool/リクエストテーブル登録用データ作成ツール.xls) | リクエストID一覧からリクエストテーブルに登録するマスタデータのテーブル投入用シートを自動生成する。 |
| [コードテーブル登録用データ出力ツール](../../../knowledge/assets/toolbox-tool/コードテーブル登録用データ出力ツール.xls) | コード設計書からコードテーブルに登録するマスタデータのテーブル投入用シートを自動生成する。 |
| [メッセージテーブル登録用データ作成ツール](../../../knowledge/assets/toolbox-tool/メッセージテーブル登録用データ作成ツール.xls) | メッセージ設計書からメッセージテーブルに登録するマスタデータのテーブル投入用シートを自動生成する。 |
| [Shell Script自動生成ツール](../../development-tools/toolbox/toolbox-ShellGenerator.md#shell-script自動生成ツール) | ジョブスケジューラから起動されるShell Scriptと環境依存項目を設定したファイルを自動生成する。 |
| [テーブル定義書及びドメイン定義書出力手順](../../../knowledge/assets/toolbox-tool/テーブル定義書およびドメイン定義書出力手順.xls) | ObjectBrowserのレポート出力機能を使ってテーブル定義書およびドメイン定義書を出力する手順を説明したドキュメント。 |
| [テーブル定義書ドメイン定義書 整合性チェックツール](../../../knowledge/assets/toolbox-tool/テーブル定義書_ドメイン定義書_整合性チェックツール.xls) | テーブル定義書記載のドメインがドメイン定義書に定義されていることや、桁数、データ型の整合性をチェックする。 |
