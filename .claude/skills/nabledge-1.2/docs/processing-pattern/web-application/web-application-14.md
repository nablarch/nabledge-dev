## Firefoxで2重サブミット防止が効かない問題

Firefoxの古いバージョン（3.x系を含む）で、`n:submitLink` で生成されたリンクを素早くダブルクリックすると、まれに2回連続でリクエストが送信される事象が発生する。Chrome/Safari/IE6 では発生しない。

**解消方法**: Firefoxを比較的最近のバージョン（9.01以降）にアップデートすることで解消する。

> **重要**: 二重サブミット防止機能は、サーバサイドのトークンチェックと併用することが前提。本事象が発生しても業務処理が2重実行されることはない。
