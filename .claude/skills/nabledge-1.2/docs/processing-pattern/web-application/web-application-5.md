# codeSelectやselectタグで、先頭に空白要素を追加する方法を教えてください

> **question:**
> codeSelectタグやselectタグを使用した場合に、先頭要素に空白要素を追加したいのですがどのように実装したらいいのでしょうか?
> また、空白要素を選択した場合、リクエストパラメータの値としては何が送信されるのでしょうか?

> **answer:**
> withNoneOptionを使用することにより、空白要素を追加することができます。
> また、空白要素を選択した場合には、空文字列("")がリクエストパラメータの値として送信されます。

> withNoneOptionの詳細は、codeSelect及びselectタグのドキュメントを参照してください。
