# codeSelectやselectタグで、先頭に空白要素を追加する方法を教えてください

## codeSelect/selectタグへの空白要素追加

`codeSelect`タグや`select`タグで先頭に空白要素を追加するには、`withNoneOption`を使用する。

- 空白要素を選択した場合、リクエストパラメータの値として空文字列（`""`）が送信される。
