## 想定外レコード識別子が存在した場合の動作

## 想定外レコード識別子が存在した場合の動作

ファイル読み込み処理において、フォーマット定義ファイルに定義されていない識別子の値がファイルに存在した場合、例外を送出して即座に処理を終了する。

> **重要**: 例外が送出されるため、バッチの処理は異常終了となる。FATALログが出力され、0以外の値を終了コードとしてJavaプロセスが終了する。
