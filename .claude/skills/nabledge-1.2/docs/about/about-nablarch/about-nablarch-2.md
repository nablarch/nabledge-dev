# configファイルの設定値の取得方法を教えてください

## configファイルの設定値の取得方法

**クラス**: `SystemRepository`

configファイルの値は`SystemRepository`クラスの`getString`メソッドで取得する。

非アスキー文字（マルチバイト文字など）はpropertyファイルのようにユニコードエスケープで設定する必要はなく、そのまま設定可能。
