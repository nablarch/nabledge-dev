# 精査エラーのメッセージを、任意の項目に対応させて画面表示させる方法を教えてください

> **question:**
> データベースアクセスを必要とする精査を行った場合(EntityやForm以外の箇所で精査を行った場合)のエラーメッセージを、
> 任意の画面項目と対応させて画面表示させる方法が分かりません。

> **answer:**
> *ValidationUtil.createMessageForProperty* を使用すると、エラー発生箇所を示すプロパティ名をメッセージの属性に追加できるため、
> エラーメッセージを特定の項目に対応させて表示させたり、画面の任意の箇所にメッセージを表示させたりすることができます。

> 詳細は、以下のドキュメントを参照してください。
