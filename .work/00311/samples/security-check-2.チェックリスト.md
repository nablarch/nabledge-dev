# 2.チェックリスト

IPAで公開されている脆弱性の種類ごとにNablarchでの対応状況を以下に記載します。
Nablarchで対応できないものについては、プロジェクトで個別に対応を検討してください。
IPAで公開されている脆弱性対策で根本的解決となっているものについては必ず対応してください。各対策に対応するNablarchの機能を右に示します。
保険的対策については対応要否をシステム要件に合わせて判断してください。
チェックに※がついているものについては、実施項目のいずれかを実施する。対応するNablarchの機能が複数実施項目にある場合はいずれで対応しても問題ない。
