お客様はNablarchのコンテンツを使用、複製、改変・翻案することができます。ただしコンテンツ種類により、以下の制限があります。

|                           |                                                                 |                                                  |

|Nablarchツール             |                                                                 |                                                  |

.. [1] Nablarchフレームワーク（アプリケーションフレームワークおよびテスティングフレームワーク）の更新バージョンは、後方互換性が維持されます。
