.. include:: ../api/link.rst

====================================================
応答不要メッセージ送信常駐バッチ
====================================================
特定のテーブル上の内容を定期的に監視し、各レコードの内容をもとにメッセージを作成して送信する常駐バッチ。
応答を伴わないメッセージ送信処理で用いる。

利用側からは対象テーブルへのINSERT文を発行するだけでメッセージを送信することができる。


++++++++++++++++

-------------------------------------------
基本構造
-------------------------------------------

:doc:`messaging_sending_batch` の構成は通常の :doc:`../architectural_pattern/batch_resident_thread_sync` に対し、
メッセージ送信処理を行う以下のハンドラを追加したものである。

  1. :doc:`../handler/MessagingContextHandler`
  2. :doc:`../handler/AsyncMessageSendAction`

本バッチでは業務アクションハンドラを新たに作成する必要は無く、
:doc:`../handler/AsyncMessageSendAction` に対する各種設定および、このハンドラ内で使用するSQL文及び、
送信電文のフォーマット定義を作成すればよい。

作成が必要なSQL文とフォーマット定義の一覧は以下のとおり。
  
  +-------------------------------------------+-----------------------------------------------------------+
  | 内容                                      | リソース名                                                |
  +===========================================+===========================================================+
  | 送信要求テーブルから送信対象の            | (SQL定義配置パッケージ設定値).                            |
  | レコードを取得するクエリのSQL             |  (メッセージリクエストID)#                                |
  |                                           |  SELECT_SEND_DATA                                         |
  +-------------------------------------------+-----------------------------------------------------------+
  | 送信完了後、対象レコードの処理ステータス  | (SQL定義配置パッケージ設定値).                            |
  | を"送信済み"に変更するSQL文               |  (メッセージリクエストID)#                                |
  |                                           |  UPDATE_NORMAL_END                                        |
  +-------------------------------------------+-----------------------------------------------------------+
  | 処理中にエラーが発生した場合に、          | (SQL定義配置パッケージ設定値).                            |
  | 対象レコードの処理ステータス              |  (メッセージリクエストID)#                                |
  | を"エラー"に変更するSQL文                 |  UPDATE_ABNORMAL_END"                                     |
  +-------------------------------------------+-----------------------------------------------------------+
  | フレームワーク制御ヘッダ部のフォーマット  | (フォーマット定義ファイル配置先論理パス名)/               |
  | 定義ファイル                              | (フレームワーク制御ヘッダフォーマットファイル名)          |

  | ファイル                                  | (メッセージリクエストID)_SEND                             |
  +-------------------------------------------+-----------------------------------------------------------+

各種設定値については、 :doc:`../handler/AsyncMessageSendAction` を参照すること。
